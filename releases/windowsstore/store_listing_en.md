Short description:
Local backup and sync tool with database safety, batch runs, and redacted profile exchange.

Description:
ProSync is a local desktop application for file and folder synchronization with a strong focus on safe backups for SQLite and Access data. It works directly with local paths or network shares and makes recurring sync jobs transparent and repeatable.

- Folder synchronization for local paths and network shares
- File synchronization for individual files and databases
- SQLite/Access safety with deliberate handling of critical cases
- Batch sync for multiple selected connections
- Scheduled backups running in the tray
- Local sync reports with counters and timestamps
- Redacted `prosync-profile-v1.json` export without real local paths or secrets

Audience:
Developers, freelancers, power users, and small teams who want to back up or synchronize project folders, documents, or database files across devices.

Keywords:
backup, sync, file sync, folder sync, SQLite, database safety, batch sync, network share, profile export, local backup
