# Support

## Deutsch

- GitHub Issues: <https://github.com/file-bricks/ProSync/issues>
- Sicherheitslücken bitte nicht öffentlich melden. Nutze dafür GitHubs private Vulnerability-Reporting-Funktion im Tab **Security**.
- Bei Bugmeldungen sind diese Angaben hilfreich:
  - ProSync-Version
  - Windows-Version
  - Verbindungsart (`folder` oder `file`)
  - Verwendeter Sync-Modus
  - Reproduzierbare Schritte
  - Relevante Ausschnitte aus lokalen Logs oder Sync-Reports ohne private Pfade

## English

- GitHub Issues: <https://github.com/file-bricks/ProSync/issues>
- Do not report security vulnerabilities publicly. Use GitHub's private vulnerability reporting flow in the **Security** tab.
- Helpful bug report details:
  - ProSync version
  - Windows version
  - Connection type (`folder` or `file`)
  - Active sync mode
  - Reproducible steps
  - Relevant local log or sync-report excerpts without private paths
